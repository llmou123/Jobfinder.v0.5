# Jobfinder.v0.5

job search application that allows users to create profiles, showcase their skills and experience, and connect with recruiters.



🛠 Tech Stack

Language: Java 

Integrated Development Environment (IDE): Android Studio

Database & Authentication : Google FireBase


✨ Features

Build professional profiles (skills, experience, education)

Search and apply for job listings

Recruiters can post job opportunities

Direct messaging between recruiters and candidates

Profile and company browsing





*This project was originally developed in 2023 and later uploaded to GitHub!*
